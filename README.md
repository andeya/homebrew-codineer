# Homebrew Tap for Codineer

Homebrew formulae for [Codineer](https://github.com/andeya/codineer) — your local AI coding agent.

## Install

```bash
brew install andeya/codineer/codineer
```

## Update

```bash
brew upgrade codineer
```

## About

This tap is automatically updated by the Codineer release CI whenever a new version is tagged.
